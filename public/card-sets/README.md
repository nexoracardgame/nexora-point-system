Put CARD SET cover images in this folder.

File names must match the set order:

- 1.png = CARD SET 1
- 2.jpg = CARD SET 2
- 3.webp = CARD SET 3

Supported extensions, checked in this order:

1. png
2. jpg
3. jpeg
4. webp

Use one image per set number. The app centers each image and keeps it fully visible inside the card frame.
